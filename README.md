# 📊 Goal Tracker Dashboard

Goal Tracker Dashboard — это веб-приложение для отслеживания целей, ежедневной активности и анализа прогресса за неделю.

Приложение помогает пользователю:

- фиксировать выполненные действия
- отслеживать прогресс по целям
- анализировать активность по неделям
- поддерживать серию (streak) выполнения

# 🚀 Основной функционал

## 🗓 Weekly Calendar

- Отображение текущей календарной недели (Mon–Sun)
- Подсветка текущего дня
- Возможность переключения между неделями
- Отображение выполненных действий по дням
- Визуальная индикация активности

## 🎯 Goal Management

- Создание целей
- Привязка действий к конкретной цели
- Отображение прогресса выполнения цели
- Подсчёт статистику выполнения за выбранную неделю

## 📈 Weekly Analytics

- Аналитика рассчитывается в рамках выбранной недели
- Топ целей по активности
- Прогресс выполнения за период
- Возможность просмотра предыдущих недель

## 🔥 Streak System

- Подсчёт серии дней с активностью
- Автоматическое обновление при добавлении действий
- Визуальное отображение текущего streak

## 📊 Progress Visualization

- Графическое отображение активности
- Сравнение результатов внутри недели
- Отображение распределения активности по дням
